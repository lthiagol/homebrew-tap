# awesome-stow

TUI wrapper for GNU Stow — manage your dotfiles interactively.

- Homepage: https://github.com/lthiagol/awesome-stow
- Tap formula: `lthiagol/tap/awesome-stow`

## Requirements

- [GNU Stow](https://www.gnu.org/software/stow/) (installed automatically as a dependency)
- Rust (installed automatically as a build dependency)

## Usage

```bash
brew install lthiagol/tap/awesome-stow
brew upgrade lthiagol/tap/awesome-stow
```
